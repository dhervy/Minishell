# Minishell

Le but est de créer un shell qui exécute quelques commandes simples et quelques bultins.

## Compiling

En exécutant `make`, un exécutable nommé `minishell` sera compilé directement.


## Usgae

`~ ./minishell`

```
$> ls
$> echo "$PATH"
$> env
$> cd ../test
$> cd -
$> mkdir
$> chmod 777 ../test/fichier
```

## Bultin

* cd
* echo
* env
* setenv
* unsetenv
