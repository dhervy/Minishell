# Minishell

Le but était de crée un shell qui executait des commande simple et quelque bultin.

## Compiling

executer `make`, un exécutable nommé `Minishell` devrait compiler directement.

## Usgae

`~ ./minishell`

```
$> ls
$> echo "$PATH"
$> env
$> cd ../test
$> cd -
$> mkdir
$> chmod 777 ../test/fichier
```

## Bultin

* cd
* echo
* env
* setenv
* unsetenv
